Powerful Features

Advanced Filters

Filter by playtime, genre, achievements, status, size, friends, tags and much more with AND/OR logic.

Multiple Sources

Combine collections, tabs, wishlist and store into a single shelf via Union or Intersection.

Configurable Shortcuts

Remap or disable the gamepad buttons for hide, highlight, quick-launch, Quick Search and Side Navigation.

Decoration Cards

Pin text, banners, URL shortcuts or gaps — with your own chosen art that can act as the shelf hero.

Smart Shelves

30+ heuristic shelves like Deck Picks or Never Played that appear automatically when relevant.

Profiles & Backup

Save, switch, export and import your setups. Includes a Default profile to reset anytime.

Installation

Deck Shelves runs on top of Decky Loader. Install it from the Decky Store, or grab the latest release and add it manually.

From Decky Store

The easiest way — once installed, Deck Shelves updates on a click of a button whenever a new version is available in the store.

  1. Install Decky Loader on your Steam Deck.
  2. Open the Decky Store and search for Deck Shelves.
  3. Install and restart Steam if prompted — future updates land on their own.

Manual installation

Install the latest build directly from GitHub Releases.

  1. Download the latest deck-shelves-v*.zip from the Releases page.
  2. In game mode, go to Decky Developer → Install from zip file.
  3. Select the zip, confirm, and restart Steam if prompted.
